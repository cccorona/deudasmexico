# PRD - Landing Page Resolución de Deudas México

## Problema Original
Crear landing page minimalista en español (México) para personas con problemas de deudas, buró de crédito o cobranza. Objetivo: conversión hacia chat de asistente automático.

## Arquitectura
- **Frontend**: React con Tailwind CSS
- **Backend**: FastAPI (no usado para esta landing)
- **Base de datos**: MongoDB (no usado para esta landing)

## User Personas
1. **Persona con deuda activa**: Recibe llamadas de cobranza, no sabe si debe pagar
2. **Persona en Buró de crédito**: Quiere limpiar su historial
3. **Persona con cargos no reconocidos**: Necesita verificar validez
